About
=========

Magento ajax multiselect filter module.


Install via Composer
=========

Add to your `composer.json`

```
"require": {
    "oggetto/filter": "*"
},
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/deniswhite8/Magento_Filter"
    }
],
"extra": {
    "magento-root-dir": ".",
}


```
